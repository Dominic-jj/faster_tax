# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

This is a document template directory for Chinese tax invoices (发票/fapiao). It contains Excel templates and reference images, not application source code.

## Contents

- `invoice_template.xlsx` — Invoice template for generating standardized fapiao documents
- `tax_list.xlsx` — Tax item listing template
- `test.jpg` — Reference image (likely a sample invoice or styling guide)

## Working with Templates

Templates are `.xlsx` files that should be opened and edited with Microsoft Excel or a compatible spreadsheet application. When modifying templates, preserve the existing formatting structure as it aligns with China's fapiao system requirements.
