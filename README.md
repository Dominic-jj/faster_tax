# 发票识别转换工具

上传发票图片或手动输入物品信息，自动匹配税收分类编码，生成标准电子发票明细 xlsx 文件。

## 快速开始

### Windows

双击 `start.bat`

### Linux / Mac

```bash
bash start.sh
```

### 手动启动

```bash
pip install -r requirements.txt
streamlit run app.py
```

浏览器自动打开 `http://localhost:8501`

## 功能

- **图片 OCR 识别** — 上传 jpg、jpeg、png 发票图片，自动提取文本行并尝试解析物品信息
- **手动输入** — 直接粘贴物品列表，格式：`规格 物品名 单位 数量 单价`
- **规格型号提取** — 自动从物品名前缀提取（如 `2P 324空气开关` → 规格 `2P 324`）
- **单位识别** — OCR/文本解析时自动从物品名后缀提取单位，支持扎、个、只、米、箱等常用单位
- **税收分类匹配** — 自动匹配税收分类编码，并支持在界面中手动修正
- **明细表编辑** — 识别结果可直接修改、增删行后再导出
- **一键生成 xlsx** — 基于 `data/invoice_template.xlsx` 模板输出，税率 0.01，编码以文本格式写入防止科学计数法

## 输入格式

### 手动输入

每行一个物品，空格分隔：

```
规格 物品名 单位 数量 单价
```

无规格时用 `-` 代替。示例：

```
16A 插座 个 4 15
- 公牛有线 14座 组 4 38
2P 空气开关 个 3 30
30X60 平板灯 个 2 50
- 热水器 台 1 37
- 时控开关 个 1 55
- 电线 米 10 5
```

### OCR / 文本解析

OCR 识别后的文本行会交给解析器自动提取物品信息，支持以下形式：

```text
物品名 数量 单价
物品名 数量 单价 总价
```

示例：

```text
插座 4 15
16A 插座 4 15 60
公牛有线 14座 4 38 152
2P 324空气开关 3 30 90
30X60 平板灯 2 50 100
```

说明：

- 当只识别出数量和单价时，总价会自动计算。
- 当文本中带有规格前缀时，会自动拆分到“规格型号”字段。
- 当物品名末尾带有常见单位时，会自动拆分到“单位”字段；否则默认按“个”处理。
